## Nextjs-boilerplate
Next.js + Tailwindcss + MSW 초기설정을 한 보일러플레이트입니다.

각 버전은 다음과 같습니다.
```json
    "msw": "^0.43.1",
    "next": "12.2.0",
    "tailwindcss": "^3.1.4",
```
